HTML & CSS Table Reference

HTML Tags

Tag Name Description
<!DOCTYPE> Document Type This declaration shoould always be the first line of code within your HTML file, as it lets the browser know what version of HTML to expect.
<html> HTML This element will contain all of your HTML code.
<head> Head All of the information about the webpage, like the title, beloongs within this section of the page.
<title> Title You can add a title to your webpage by using this element, inside of the head. A webpage's title appears in the browser's tab.
<a> Anchor Anchor tags are used to link to internal pages, external pages or content on the same page. You can create sections on a webpage and jump to them using anchor tags and adding ID's to the elements you wish to jump to.
<!--comment--> Comments Comments are written in HTML using this tag, to add explanatory notes to the markup or to prevent the browser from interpreting specific parts of the document.

CSS Properties

Property Name Description
color:value Text Colour Sets the colour of the text.
background-colour:value Background Colour Specifies the background colour of an element.
font-family: value Font Specifies the font famnily of the text.
font-size: value Font Size Specifies the size of the font.
text-align: value Text Positioning Specifices the horizontal alignment of the text.
margin: value Margin Sets all of the margin properties in one declaration
